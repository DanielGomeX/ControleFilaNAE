INSTALACAO
-------------------------------------------------------------------------------------------------------

NECESSÁRIO INSTALAR:
    Servidor (XAMPP ou WAMP ou separados como, apache, php e mysql)

Tendo o servidor, copie essa pasta completa para a pasta do seu servidor.

- No caso do XAMPP, coloque-a dentro de htdocs localizada dentro da pasta do XAMPP

- No caso do WAMP, coloque-a dentro de www localizada na pasta do WAMP

- Se for separados pode colocar em www, provavelmente localizada na sua pasta do apache
--------------------------------------------------------------------------------------------------------

BANCO DE DADOS

Após instalar o necesário, copiar a pasta, pegue o script do banco de dados e execute-o, criando assiim o
banco de dados e tabelas necessárias para a execucao correta do sistema

Fique responsável por inserir os funcionários que irão trabalhar como sistema
--------------------------------------------------------------------------------------------------------

Após isso instalado, acesse o vídeo para ver a utilização do sistema

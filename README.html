Sistema de Controle de Fila do NAE

O Sistema de Controle de Fila do NAE foi criado por uma equipe do 5° semestre do curso de Análise e Desenvolvimento de Sistemas da Faculdade São Salvador para o projeto de estágio

Abaixo deixamos o passo a passo e coisas necessárias para a implantação do sistema

Passo 1 : Instalação de dependências

Para que o sistema funcione perfeitamente, precisará ter um servidor na sua máquina.
Pode instalar separadamente as seguintes dependencias:

MySQL
Apache
PHP

Pode utilizar softwares completos como: (Recomendável)

XAMPP
WAMP Server

Se o seu sistema for LINUX, acesse um dos links abaixo para instalar o LAMP na sua máquina

Instale o LAMP no Linux e tenha um servidor web no PC
Como instalar o LAMP no Ubuntu 18.04

Após isso, baixo o MySQL Workbench para gerenciar o banco de dados do sistema
Estamos disponibilizando o MySQL Workbench que é um software muito completo, mas pode utilizar outro da sua preferência

MySQL Workbench

Passo 2: Criando banco e adicionando usuários

Com o primeiro passo concluido, inicie o MySQL Workbench para executar os scripts necessários

Nessa mesma pasta onde o arquivo README.html está, existe dois scripts de nomes

script para criar banco de dados.sql
script para inserir usuários.sql

Abra o arquivo e execute o codigo no MySQL Workbench.
No script para inserir usuários existe uma explicação básica informando para digitar algo como:

INSERT INTO tb_nae VALUES(null, 'Nome Completo', 'Usuario', 'Senha')

Pode substituir onde diz Nome Completo, Usuario e Senha pelos dados do usuário e executar

Passo 3: Abrindo o sistema

Copie a pasta NAE onde está todo o sistema para a pasta do seu servidor

Se for o XAMPP copie para a pasta do caminho 'C:\xampp/htdocs'
Se for o WAMP copie para a pasta do caminho 'C:\wamp64\www'

Passo 4: Executando

Abra o navegador e digite o seguinte endereço: localhost/NAE

Dessa maneira você já estará no sistema de controle, basta escolher a opção desejada
